# Autonomous Optical Detector for MEMS Characterization

Designed and developed an autonomous optoelectronic detection system for MEMS characterization, integrating custom PCB circuitry, a Raspberry Pi 4, and a Keysight Function Generator. Engineered a closed-loop automation architecture using Python, PyVISA, and SCPI protocols to execute vibration-free 0-300 Hz frequency sweeps. 

Simultaneously, developed a 120 FPS real-time Computer Vision pipeline with OpenCV to precisely track the MEMS mirror's optical scan angle, overcoming hardware motion blur constraints by locking exposure time and optimizing analog gain. Successfully processed a dataset of over 55,000 rows, achieving a highly stable sub-milligram mass detection sensitivity of 0.38 Hz/mg, demonstrating the system's viability for biomolecular Point-of-Care diagnostics.
